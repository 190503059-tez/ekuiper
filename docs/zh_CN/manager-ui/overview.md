## Kuiper 管理控制台的使用

### 概览

从 Kuiper 0.9.1 版本开始，每发布一个 Kuiper 新版本，会随之发布对应版本的管理控制台。本文以一个实际例子来说明如何使用管理控制台对 Kuiper 节点进行操作与管理，主要包括以下功能：

- Kuiper 节点的增加与删除
- 插件管理
- 流管理
- 规则管理
